Política de Privacidade do Aplicativo Milvan

Última atualizaçăo: 24/06/2026

Esta é a política de privacidade oficial do aplicativo Milvan para Android, pacote com.mils.app.aluno. Ela descreve como coletamos, usamos, retemos e excluímos dados pessoais dos usuários.

1. Dados coletados

  • Câmera: usada apenas para capturar imagens necessárias ao login e ŕ autenticaçăo biométrica. Essas imagens năo săo armazenadas de forma permanente no servidor.
  • Dados biométricos: utilizados exclusivamente para autenticaçăo local no dispositivo. Esses dados năo săo enviados a terceiros ou armazenados em servidores externos.
  • Dados de conta e login: informaçőes necessárias para autenticar o usuário e permitir o acesso ao app.
  • Dados de uso: informaçőes sobre o uso do app e preferęncias podem ser armazenadas para melhorar a experięncia do usuário.

2. Finalidade do uso

  • Permitir acesso seguro ao aplicativo.
  • Autenticar o usuário aluno.
  • Manter preferęncias e configuraçőes para melhor experięncia.
  • Garantir a operaçăo do aplicativo de forma segura.

3. Retençăo de dados

Os dados săo retidos enquanto o usuário utiliza o aplicativo ou enquanto a conta estiver ativa.

Os dados armazenados localmente no dispositivo permanecem até que o app seja desinstalado ou até que o usuário solicite exclusăo.

Se houver armazenamento em servidor, os dados săo mantidos enquanto a conta existir e săo excluídos em até 30 dias úteis após solicitaçăo de exclusăo.

4. Exclusăo de dados

  • Para excluir dados locais, o usuário pode desinstalar o aplicativo do dispositivo.
  • Para excluir dados vinculados ŕ conta ou a histórico em servidor, o usuário deve solicitar exclusăo pelo email abaixo.
  • Após o pedido de exclusăo, o processo será concluído em até 30 dias úteis.

5. Compartilhamento de dados

  • Nenhum dado é compartilhado com terceiros sem o consentimento explícito do usuário.
  • Dados biométricos permanecem no dispositivo e năo săo transferidos para outros serviços.

6. Segurança

  • Utilizamos técnicas de proteçăo adequadas para assegurar dados sensíveis.
  • Năo armazenamos imagens ou dados biométricos em servidores externos sem consentimento.

7. Direitos do usuário

  • O usuário pode solicitar acesso, correçăo ou exclusăo de seus dados a qualquer momento.
  • Dados locais podem ser excluídos com a desinstalaçăo do app.
  • Dados armazenados em servidor podem ser excluídos mediante solicitaçăo por email.

Para solicitar exclusăo de dados ou esclarecer dúvidas sobre esta política, entre em contato:

milscodgos@gmail.com

Ao solicitar exclusăo, informe seu nome, email e, se possível, o identificador do usuário no app.

8. Alteraçőes nesta política

Esta política pode ser atualizada. A data acima será alterada em caso de mudanças relevantes.
