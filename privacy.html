Política de Privacidade do Aplicativo Milvan

Última atualização: 24/06/2026

Esta é a política de privacidade oficial do aplicativo Milvan para Android, pacote com.mils.app.aluno. Ela descreve como coletamos, usamos, retemos e excluímos dados pessoais dos usuários.

1. Dados coletados

  • Câmera: usada apenas para capturar imagens necessárias ao login e à autenticação biométrica. Essas imagens não são armazenadas de forma permanente no servidor.
  • Dados biométricos: utilizados exclusivamente para autenticação local no dispositivo. Esses dados não são enviados a terceiros ou armazenados em servidores externos.
  • Dados de conta e login: informações necessárias para autenticar o usuário e permitir o acesso ao app.
  • Dados de uso: informações sobre o uso do app e preferências podem ser armazenadas para melhorar a experiência do usuário.

2. Finalidade do uso

  • Permitir acesso seguro ao aplicativo.
  • Autenticar o usuário aluno.
  • Manter preferências e configurações para melhor experiência.
  • Garantir a operação do aplicativo de forma segura.

3. Retenção de dados

Os dados são retidos enquanto o usuário utiliza o aplicativo ou enquanto a conta estiver ativa.

Os dados armazenados localmente no dispositivo permanecem até que o app seja desinstalado ou até que o usuário solicite exclusão.

Se houver armazenamento em servidor, os dados são mantidos enquanto a conta existir e são excluídos em até 30 dias úteis após solicitação de exclusão.

4. Exclusão de dados

  • Para excluir dados locais, o usuário pode desinstalar o aplicativo do dispositivo.
  • Para excluir dados vinculados à conta ou a histórico em servidor, o usuário deve solicitar exclusão pelo email abaixo.
  • Após o pedido de exclusão, o processo será concluído em até 30 dias úteis.

5. Compartilhamento de dados

  • Nenhum dado é compartilhado com terceiros sem o consentimento explícito do usuário.
  • Dados biométricos permanecem no dispositivo e não são transferidos para outros serviços.

6. Segurança

  • Utilizamos técnicas de proteção adequadas para assegurar dados sensíveis.
  • Não armazenamos imagens ou dados biométricos em servidores externos sem consentimento.

7. Direitos do usuário

  • O usuário pode solicitar acesso, correção ou exclusão de seus dados a qualquer momento.
  • Dados locais podem ser excluídos com a desinstalação do app.
  • Dados armazenados em servidor podem ser excluídos mediante solicitação por email.

Para solicitar exclusão de dados ou esclarecer dúvidas sobre esta política, entre em contato:

milscodgos@gmail.com

Ao solicitar exclusão, informe seu nome, email e, se possível, o identificador do usuário no app.

8. Alterações nesta política

Esta política pode ser atualizada. A data acima será alterada em caso de mudanças relevantes.
